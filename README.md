# 🦖 Lógica de Programação: Lista 3 - Algoritmos e Flowgorithm 🖥️

![Status](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen?style=for-the-badge)

Esta é a terceira atividade sobre Lógica de Programação do curso **Técnico em Desenvolvimento de Sistemas**. Desta vez o trabalho foi ainda mais estruturado utilizando o programa Flowgorithm, junto ao VisuAlg.

O desafio consistiu na criação de sistemas pelo Flowgorithm, mas agora com enfoque no "if", na **tomada de decisões dentro do código**.

## ⚠️ Regras Importantes 

> **Atenção aos Detalhes:** Lógica é precisão, se o passo a passo conter erros, o algoritmo falha. Não esquecer de separar os tipos de dados e atenção aos comandos de **decisão**.
> **Estrutura:** Além de estrutura ainda mais complexa, utilizando a ferramenta Flowgorithm, os desafios trouxeram mais foco para o sistema de decisão dentro do código.

---

## 📂 Estrutura do Repositório

* `fluxogramas`: Representações visuais do fluxo de cada tarefa.
* `pseudocodigos`: Arquivos contendo a lógica escrita de forma estruturada.

---

## 🛠️ Ferramentas Utilizadas

* **[Flowgorithm](http://www.flowgorithm.org/):** ferramenta gráfica que permite criar e executar programas através de **fluxogramas**.
* **[Visualg](https://visualg.com.br/):** Para a implementação dos algoritmos em **Portugol (.alg)**, permitindo a simulação real dos cálculos e desvios condicionais.
